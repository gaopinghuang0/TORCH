## Welcome to gaoping-blog
Gaoping's personal blog, built on the MEAN stack.

### Tech Stack
- MEAN stack (MongoDB, Express, Angularjs, Nodejs)
- Jade template
- SASS
- jQuery
- Bootstrap
- Grunt

### Install
```bash
npm install  #back-end
bower install  #front-end
```

### Run
```shell
npm install -g grunt-cli
mongod
grunt
```
